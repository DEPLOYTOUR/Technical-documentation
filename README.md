# DEPLOYTOUR ETDS – Technical Documentation

This repository contains the technical documentation of the **DEPLOYTOUR – European Tourism Data Space (ETDS)** project.

## Contents
- Project architecture  
- MVP components and technical roadmap  
- Connector configuration & integration guidelines  
- Metadata and catalogue specifications (DCAT-AP, etc)  
- Vocabularies, ontologies, and semantic models  
- API documentation and examples  
- User interface guidelines and data collection wizard  
- Pilot onboarding documentation  

## Purpose
To provide a single, public reference point for developers, use cases and stakeholders participating in the DEPLOYTOUR ETDS initiative.

## Audience
- Technical partners  
- Use cases
